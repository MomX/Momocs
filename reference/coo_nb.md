# Counts coordinates

Returns the number of coordinates, for a single shape or a Coo object

## Usage

``` r
coo_nb(coo)
```

## Arguments

- coo:

  `matrix` of `(x; y)` coordinates or any
  [Coo](http://momx.github.io/Momocs/reference/Coo.md) object.

## Value

either a single numeric or a vector of numeric
