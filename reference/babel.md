# Convert (x; y) coordinates to chaincoded coordinates

Useful to convert (x; y) coordinates to chain-coded coordinates.

## Usage

``` r
pix2chc(coo)

chc2pix(chc)
```

## Arguments

- coo:

  (x; y) coordinates passed as a matrix

- chc:

  chain coordinates

## Value

a matrix or a numeric

## Note

Note this function will be deprecated from Momocs when `Momacs` and
`Momit` will be fully operationnal.

## References

Kuhl, F. P., & Giardina, C. R. (1982). Elliptic Fourier features of a
closed contour. *Computer Graphics and Image Processing*, 18(3),
236-258.
