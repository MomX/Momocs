# Data: Outline coordinates of mosquito wings.

Data: Outline coordinates of mosquito wings.

## Format

A [Out](http://momx.github.io/Momocs/reference/Out.md) object with the
126 mosquito wing outlines outlines used Rohlf and Archie (1984). Note
that the links defined here are quite approximate.

## Source

Rohlf F, Archie J. 1984. A comparison of Fourier methods for the
description of wing shape in mosquitoes (Diptera: Culicidae).
*Systematic Biology*: 302-317.
